# 西安电子科技大学硕士学位论文中期考核报告表（学术学位）LaTeX 模板

本项目采用严格复刻模式：`main.tex` 直接复用学校原始 PDF 模板
`西安电子科技大学硕士学位论文中期考核报告表（学术学位）.pdf`，
以保证空白模板的页面尺寸、字体、字距、框线和排版位置与原模板视觉一致。

## 编译

```bash
xelatex main.tex
```

编译后生成 `main.pdf`，共 6 页 A4。

## 文件说明

- `main.tex`：编译入口。
- `xidian-midterm.cls`：严格复刻类文件，使用 `pdfpages` 输出原模板。
- `西安电子科技大学硕士学位论文中期考核报告表（学术学位）.pdf`：权威模板底图，不能删除或改名。
- `info.tex`：历史填写接口文件，严格复刻模式下不参与编译。

## 说明

完全用 LaTeX 手工绘制 Word 模板很难做到像素级一致，原因包括 Word 字距、
字体替换、行框高度和 PDF 渲染路径差异。当前方案优先保证空白模板复刻准确。

如后续需要“可填写版本”，建议在此严格模板底图上叠加填写内容，而不是重新手绘模板框线。
